Welcome to Plutarch

Plutarch is a semantic search engine designed to find related people, places, or events, based on the related words used to describe the historical entities you are most interested in. Want to find new podcasts on malelovent dictators? Try searching 'Caligula'. Want to learn more about famous ambushes? Try searching 'Battle of the Teutoburg Forest'. Whatever you're inerested in, find new content to keep on learning!
